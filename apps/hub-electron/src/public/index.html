Setup

Follow the steps in order. Business days switch automatically at 6 AM IST.

Step 1

Unlock Hub

Create or enter the Manager PIN for this PC. Setup unlocks only after the PIN is verified.

Step 2

Business Day

The hub automatically uses a 6 AM to 6 AM IST business day. Staff do not press a daily start or end button.

Step 3

Choose Cash Counter Printer

This printer is used for final bills and payment receipts.

Fresh installs start in Test Mode. Switch to Live Mode only after real test prints work.

Advanced: LAN fallback
Step 4

Add Kitchens / Bar Counters

Add each place that prepares items. Dishes will print only at the selected kitchen or counter.

Advanced: custom ID or LAN fallback
Step 5

Add Tables

Create floors first, then add the tables waiters will select during service.

Advanced: custom floor ID
Advanced: custom table ID
Step 6

Add Menu Items

Add a normal dish, its price, and optionally choose the kitchen or counter that prepares it.

Advanced: custom dish ID
Step 7

Pair Devices

Create a QR code for a waiter, kitchen, captain, or admin device.

Ready For Service
